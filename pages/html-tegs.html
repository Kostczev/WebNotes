Формы

form

Тег form cлужит для передачи данных от пользователя. Все теги, относящиеся к форме, необходимо указывать в его теле.

Атрибуты:

  • action - отвечатет за то, куда полетят данные. (адрес сервера) Стоит заметить, что, при работе с версткой оставлять его пустым нельзы, это может привести к ошибке! (Ставится заглушка "#")
  • method - метод передачи данных.
    • Значение "get" является значением по умочанию. Данные передаются прямо в адресной строке. Они видны любому пользователю и могут использоваться неоднократно. Подходит для малого кол-ва данных.
    • Значение "post" передает данные скрытно используя запросы браузера. Позволяет отправлять большие объемы данных. В случае работы с файлами используется именно этот метод.
  • enctype со значением "multipart/form-data"

input

  • type - определяет метод приема данных от пользователя
    • По умочанию используется text
    • Для удобного ввода телефона на мобильных устройствах используется tel. (но почему-то на пк фарматирования никакого нет, смотрим дальше)
    • password - при вводе получаем звездочки
    • radio - выбор одного из. Для создания отдельныйх выборок необходимо задавать разные имена. Для задания выбора по умолчанию нобходимо добавить атрибут checked к нужному radio

  • placeholder - серый текст-подсказка на полях

textarea

Используется для ввода многострочного текста.

Атрибуты cols и rows отвечают за размер полученного элемента. Также, у пользователя имеется возможность изменить размер элемента. При верстке, такие вещи в большенстве случае нужно отключать.

select и option

нужно пофиксить отсутствие центровки у данного тега

Выпадающий список, где select его коробка а option's его содержимое. Для назначения выбора по умолчанию используется атрибут selected

Для того, чтобы создать список с выбором нескольких знаечние дабавляется атрибут multiple. Через зажатый ctrl можно выбрать несколько значений.

button

Содержит определяющий атрибут type в котором используются значчения:

  • submit - отправляет форму целиком (все заполненные поля)
  • reset - очистка формы
